sequenceDiagram
      participant U as User
      participant F as Frontend
      participant E as Express
      participant P as PostgreSQL

      U->>F: Types in the destination
      F->>F: Waits for button click or enter
      U->>F: Clicks the button or enter
      F->>E: fetch request
      E->>P: SELECT statement
      P-->>E: Rows
      alt Rows found
        E-->>F: Replies with JSON
        F-->>U: Displays data from JSON
      else No rows found
        E-->>F: No matching destination
        F-->>U: Shows no destination
      end
